# DOCKER + PRP ИНТЕРФЕЙС

## **ШАГ 1: НАСТРОИТЬ PRP (ПАРАЛЛЕЛЬНАЯ РЕЗЕРВНАЯ СЕТЬ)**

**Зачем:** Объединить 2 сетевых интерфейса для отказоустойчивости.

```bash
# На COD-DOCKER
apt install -y bridge-utils ifenslave

# Создать bond интерфейс
modprobe bonding

nano /etc/network/interfaces
```

**Добавь:**

```bash
auto bond0
iface bond0 inet static
    address 10.10.20.200
    netmask 255.255.255.0
    bond-slaves eth1 eth2
    bond-mode active-backup
    bond-miimon 100
```

**Применить:**

```bash
ifup bond0
```
